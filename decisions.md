## 현재 기술 스택 (2026-03 기준)
- 프레임워크: Next.js + TypeScript + React 19
- DB: PostgreSQL (Neon Serverless)
- 스타일: Tailwind CSS 4
- AI: Google Gemini API (캘린더 AI 어시스턴트)
- 인증: JWT (jose) + bcryptjs, 이메일 기반 회원가입
- 배포: Vercel (자동 배포)
- 배포 URL: https://calendar-app-blush.vercel.app
- 원본 프로젝트: nepcon-sorter (V1)에서 캘린더 기능만 추출
